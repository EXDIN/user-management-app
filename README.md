# User-management-app

Basic functionality with CRUD for the user

## Libraries

-   React + Vite - Frontend
-   React-router-dom - navigations
-   RTK - State Manager
-   MUI - UI
-   React-hook-forms - work with Forms
-   ZOD - validator
-   Axios - for Api
-   Vitest - testing

## To run project

-   git clone https://github.com/EXDIN/user-management-app.git
-   npm install

## Commands

-   npm start - Run App
-   npm test - Run Tests
